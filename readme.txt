Задание по Android
Мобильное приложение, представляющее из себя
ежедневник.
В приложении присутствуют 3 экрана - Авторизация, Список заметок с календарем,
Подробное описание заметки, создание заметки.

1)Список дел с календарем - экран на котором присутствует возможность
выбрать один день, после выбора дня в конце экрана должна обновиться
таблица с заметками в каждой ячейке таблицы указан 1 час из дня,
также содержится кнопка "создать новую заметку"

2)Описание дела включается в себя дату, имя, описание, время, а также две кнопки - "удалить заметку" и "назад".

3)Экран создания новой заметки начинается с выбора даты. Далее указывается время строго в формате 10:00 - 11:00 (две цифры и один час разницы между временем), а также
название заметки, описание. Также две кнопки "создать заметку" и "назад".

Целью создания приложения было получение опыта использования firebase, recyclerview, customadapter.
